| 按键       | 功能                                      | 解释     |
| ---------- | ----------------------------------------- | -------- |
| `空格`     | 向下翻页                                  |          |
| `pagedown` | 向下翻页                                  |          |
| `pageup`   | 向上翻页                                  |          |
| `tab`      | 在节点之间移动, 有节点的地方, 通常以*显示 |          |
| `enter`    | 当光标在节点上时, 按下可以进入节点        |          |
| `b`        | 移动到第一个节点处                        | begin    |
| `e`        | 移动到最后一个节点处                      | end      |
| `n`        | 前往下一个节点处                          | next     |
| `p`        | 前往上一个节点处                          | previous |
| `u`        | 向上移动一层                              |          |
| `s(/)`     | 在info page当中进行查找                   | search   |
| `h`,`?`    | 显示帮助选项                              | help     |
| `q`        | 结束这次的info page                       | quit     |
