# [options]参数表

| options | 含义                               | 对应单词 |
| ------- | ---------------------------------- | -------- |
| `l`     | 长格式输出                         | long     |
| `u`     | 按用户名和启动时间的顺序来显示进程 |          |
| `j`     | 用任务格式来显示进程               |          |
| `f`     | 用树形格式来显示进程               |          |
| `a`     | 显示所有用户的所有进程             | all      |
| `x`     | 显示无控制终端的进程               |          |
| `r`     | 显示运行中的进程                   | run      |
