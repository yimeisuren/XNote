| 按键                              | 功能                     |
| --------------------------------- | ------------------------ |
| `空格`                            | 向下翻页                 |
| `pagedown`也就是`fn`+$\downarrow$ | 向下翻页                 |
| `pageup`                          | 向上翻页                 |
| `/string`                         | 向下查找string这个字符串 |
| `?string`                         | 向上查找string这个字符串 |
| `n`,`N`                           | n表示继续, N表示反向     |
| `q`                               | 结束这次的man page       |
