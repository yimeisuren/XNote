#### 第一个字符: 文件类型

| 特征 |                            类型                            |    含义     |
| :--: | :--------------------------------------------------------: | :---------: |
| `d`  |                            目录                            | `directory` |
| `-`  |                         二进制文件                         |             |
| `l`  |                         软链接文件                         |   `link`    |
| `.`  |                          隐藏文件                          |             |
| `b`  |   设备文件里面的可供存储的周边设备(可按块随机读取的设备)   |             |
| `c`  | 设备文件里面的串行端口设备, 例如鼠标, 键盘(一次性读取设备) |             |

#### 后9个字符: 文件权限

rwx : 读权限(4), 写权限(2), 执行权限(1)

第一组rwx: 文件拥有者的权限

第二组rwx:加入此用户组的成员的权限

第三组rwx:其它成员的权限

#### 注意事项

+ ==rwx权限对于文件和目录的区别: 能否删除一个文件，关键在于对于这个文件所属的目录是否有写的权限==
+ 无论文件的权限是什么, root用户都可以进行读写和执行
+ ugo: u拥有者(user), g组员(group), o其它人(other)
