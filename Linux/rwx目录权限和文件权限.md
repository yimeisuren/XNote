| 权限 | 文件                                                     | 目录                                       |
| :--- | -------------------------------------------------------- | :----------------------------------------- |
| `r`  | 读取文件内容<br />*文件最基础的权限*                     | 查看目录中文件名<br />tab补全功能的支持    |
| `w`  | 更改文件内容                                             | 对目录中文件名的增删改操作                 |
| `x`  | 决定文件是否可以执行<br />但可执行与执行成功与否是两码事 | 是否可以进入该目录<br />*目录最基础的权限* |

| 权限  | 文件 | 目录 |
| ----- | ---- | ---- |
| `r--` |      |      |
| `-w-` |      |      |
| `--x` |      |      |
| `rw-` |      |      |
| `r-x` |      |      |
| `-wx` |      |      |
| `rwx` |      |      |
| `---` |      |      |
