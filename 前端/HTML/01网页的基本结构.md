# 网页基本结构

+ `head`标签
+ `meta`标签
+ `title`标签
+ `body`标签

```html
<!DOCTYPE html>
<html lang="en">

<!--head标签: 不会出现在网页中直接出现, 主要用来帮助浏览器或搜索-->
<head>
    <!--    meta标签: 描述性标签,用来描述网页的一些信息,关键字等-->
    <meta charset="UTF-8">
    <meta name="keywords" content="html,开源">
    <meta name="description" content="学习java">
    <!--    title标签: 网页的标题-->
    <title>网站的标题</title>
</head>


<!--body标签: 网页主体-->
<body>
<!--网页文本信息-->
hello world
</body>
</html>
```
